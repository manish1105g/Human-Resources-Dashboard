# 👥 Human Resources Dashboard

## 📌 Project Overview

This project focuses on analyzing employee data to uncover insights related to workforce distribution, attrition, performance, and diversity. The dashboard provides an interactive way to explore HR trends and support strategic decision-making.

---

## 🎯 Objectives

* Analyze employee performance and productivity
* Identify attrition trends and key factors
* Understand workforce distribution across departments
* Provide insights for better HR decision-making

---

## 📊 Dashboard Features

### 👨‍💼 Employee Overview

* Total employee count
* Department-wise distribution
* Role-based insights

### 📉 Attrition Analysis

* Attrition rate overview
* Department-wise attrition
* Key factors influencing employee turnover

### 📊 Performance Insights

* Employee performance metrics
* High vs low performers
* Trends in productivity

### 🌍 Diversity Analysis

* Gender distribution
* Workforce diversity insights
* Inclusion metrics

### 📅 Trend Analysis

* Hiring trends over time
* Employee growth patterns
* Workforce changes year-wise

### 🔍 Interactive Filters

* Department
* Gender
* Job Role
* Year

---

## 🎨 Dashboard Design

* Clean and intuitive layout
* Interactive visuals for better exploration
* Focus on clarity and actionable insights

---

## 💡 Use Cases

* Identify high attrition areas
* Improve employee retention strategies
* Analyze workforce diversity
* Support HR planning and decision-making
